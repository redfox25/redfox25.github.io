About Me

Hello! I’m a QA Engineer. Right now I’m working for company called Popsugar in San Francisco. As the quality assurance (QA) engineer I make sure that all of the programs work properly and that everything in a Web site functions according to plan. In general QA engineers are the bridge between end-users and programmer. Basically I defend the end-user point of view and the same time I try and make things not to work so developers can catch issue before users see them. I also enjoy create things using the latest technoliges.
